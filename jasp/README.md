# JASP-Vorlagen

Dieser Ordner enthält JASP-Vorlagendateien (`.jasp`), die den kombinierten Datensatz bereits eingebunden haben.

## Verwendung

1. `tag2-template.jasp` für Tag 2 öffnen (Datensatz laden, Deskriptive, Reliabilität, EFA)
2. `tag3-template.jasp` für Tag 3 öffnen (Regression, Moderation, Mediation)

JASP-Dateien enthalten den **relativen Pfad** zum Datensatz (`../../data/combined/m15-combined.csv`). Stelle sicher, dass Du das Repository lokal geklont hast und der Datensatz vorhanden ist.

## Hinweis

Die `.jasp`-Dateien sind zunächst leere Vorlagen. Die Lehrperson füllt sie mit dem kombinierten Datensatz auf, sobald dieser nach Tag 1 vorliegt.
